# 🐍 Snake Game in C++

A classic, console-based implementation of the retro Snake game, written entirely in C++. This project brings the nostalgic arcade experience to your terminal, complete with score tracking and integrated sound effects!

## ✨ Features

* **Classic Gameplay:** Navigate the snake to eat food and grow longer without hitting the walls or yourself.
* **Real-time Input:** Smooth console controls for directional movement.
* **Audio Effects:** Integrated sound effects (`beep.wav`) for game events like eating food or game over states.
* **Score Tracking:** Keeps a running total of your score as the snake grows.

## 🛠️ Tech Stack

* **Language:** C++
* **Environment:** Command Line / Terminal
